# 'GeoNetwork' API R Interface

Provides an R interface to the 'GeoNetwork' API
(\<https://geonetwork-opensource.org/#api\>) allowing to upload and
publish metadata in a 'GeoNetwork' web-application and expose it to OGC
CSW Web-Services (Catalogue Service for the Web).
